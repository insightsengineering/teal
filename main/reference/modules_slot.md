# Retrieve slot from `teal_modules`

Retrieve slot from `teal_modules`

## Usage

``` r
modules_slot(modules, slot)
```

## Arguments

- modules:

  (`teal_modules`)

- slot:

  (`character(1)`)

## Value

A `list` containing the `slot` of the modules. If the modules are
nested, the function returns a nested `list` of values.
