# Does the object make use of the `arg`

Does the object make use of the `arg`

## Usage

``` r
is_arg_used(modules, arg)
```

## Arguments

- modules:

  (`teal_module` or `teal_modules`) object

- arg:

  (`character(1)`) names of the arguments to be checked against formals
  of `teal` modules.

## Value

`logical` whether the object makes use of `arg`.
