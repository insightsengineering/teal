# Retrieve `teal_bookmarkable` attribute from `teal_modules`

Retrieve `teal_bookmarkable` attribute from `teal_modules`

## Usage

``` r
modules_bookmarkable(modules)
```

## Arguments

- modules:

  (`teal_modules` or `teal_module`) object

## Value

named list of the same structure as `modules` with `TRUE` or `FALSE`
values indicating whether the module is bookmarkable.
