# Append a `teal_module` to `children` of a `teal_modules` object

Append a `teal_module` to `children` of a `teal_modules` object

## Usage

``` r
append_module(modules, module)
```

## Arguments

- modules:

  (`teal_modules`)

- module:

  (`teal_module`) object to be appended onto the children of `modules`

## Value

A `teal_modules` object with `module` appended.
